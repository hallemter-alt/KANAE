### 設定前の確認

dig www.rut-tokyo.com CNAME

# または
nslookup www.rut-tokyo.com
```

**期待される結果（設定前）**:
```

# 新しい DNS レコードを確認
dig www.rut-tokyo.com CNAME

# または
nslookup www.rut-tokyo.com

## 🎯 成功の確認方法

### ブラウザで確認
1. `https://www.rut-tokyo.com/` を開く
2. 新不動産サイトが表示される ✓
3. アドレスバーに 🔒 が表示される ✓
4. `/rent`, `/sale`, `/minpaku` が正常に表示される ✓

### コマンドで確認
```bash
# HTTP ステータスを確認
curl -I https://www.rut-tokyo.com/
# → HTTP/2 200 が返る

# DNS レコードを確認
dig www.rut-tokyo.com CNAME
# → cname.vercel-dns.com が返る

# SSL 証明書を確認
openssl s_client -connect www.rut-tokyo.com:443 -servername www.rut-tokyo.com < /dev/null 2>/dev/null | openssl x509 -noout -dates
# → 有効期限が表示される
```
