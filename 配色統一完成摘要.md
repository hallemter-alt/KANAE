## 🎯 解決的問題

用戶反饋：
> "導航欄項目點擊進去後的頁面字體顏色和背景色，請根據首頁的設計概念做統一設計。保持整體色調的統一性，還有選擇欄目的下拉選項字體也調整成深灰色或黑色。"

### 修復內容

#### 1. **Hero 區域統一** ⭐
- **問題**: 內頁使用淺色背景，與首頁風格不一致
- **解決**: 所有內頁 Hero 改為深色漸變 + 白色文字
- **效果**: 與首頁保持視覺一致性

#### 2. **下拉選項文字顏色** ⭐⭐⭐
- **問題**: select 和 option 元素沒有文字顏色，顯示淺灰色難以閱讀
- **解決**: 所有下拉選項添加 `text-gray-900`（深灰/黑色）
- **效果**: 文字清晰可讀，用戶體驗大幅提升

#### 3. **配色方案統一**
- **問題**: 使用 amber、purple、blue 等不一致的顏色
- **解決**: 統一為 primary（藍）+ gold（金）
- **效果**: 品牌視覺 100% 一致

## 🎨 統一後的配色

### Hero 區域
```css
背景: bg-gradient-to-br from-primary-900 via-primary-800 to-gold-900
標題: text-white
描述: text-white/90
```

### 表單元素
```css
Select: text-gray-900 bg-white
Option: text-gray-900
Input: text-gray-900 bg-white
聚焦: focus:ring-primary-500
邊框: border-2 border-gray-300
```

### 按鈕
```css
主按鈕: bg-gradient-to-r from-primary-600 to-primary-700
次按鈕: from-gold-600 to-gold-700
懸停: hover:from-primary-700 hover:to-primary-800
```
